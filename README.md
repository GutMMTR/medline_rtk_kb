# Медлайн.РТК.КБ — MVP хранилища файлов

## Что уже сделано
- Backend: **FastAPI + HTML UI** (таблица как в Excel).
- Auth: логин/пароль, cookie с JWT.
- Роли (MVP):
  - **admin**: всё.
  - **auditor**: в MVP считается *глобальным*, если у пользователя есть хотя бы один доступ с ролью `auditor` (видит все организации).
  - **customer**: доступ только к своим организациям.
- Docker: Postgres + backend.
- Миграции: Alembic, автоприменение при старте backend контейнера.

## Запуск
В корне проекта:

1) Поднять всё:
```bash
docker compose up --build
```

2) Открыть UI:
- `http://localhost:8000/login`

## Дефолтный админ
По умолчанию (из `docker-compose.yml`):
- login: `admin`
- password: `admin12345`

Меняется через переменные окружения `ADMIN_LOGIN`, `ADMIN_PASSWORD`, `ADMIN_FULL_NAME`.
